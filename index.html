Flexible-scrollbar

A jQuery plugin for making highly configurable scrollbars.

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by ayeressian

Comapatability

Has been tested on following browsers.

  • IE 9+
  • Chrome (Android, PC, MAC)
  • Firefox
  • Safari (MAC, IOS)

Simple example

On the left side are containers with flexible scrollbar. On the right side are containers with browser's default scrollbar.
